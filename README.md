## Rules of the build
1. **Additive only** — new files beside the site; the working product is never rebuilt.
2. **One-line installs** — widget and enhancement layer mount with a single script tag.
3. **Guard every effect** — try/catch all blocks; heavy effects off on mobile/touch; `prefers-reduced-motion` respected.
4. **Verify before shipping** — screenshot render + diff on every homepage edit.
5. **Layer, don't delete** — confused visitors get a fast lane, not a slimmer site.
